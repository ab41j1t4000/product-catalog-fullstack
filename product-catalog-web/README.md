# Product Catalog Web

Minimal React + Vite + TypeScript starter for learning how a frontend app runs.

## What to read first

1. `src/main.tsx`
2. `src/App.tsx`
3. `src/index.css`

## Run locally

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5173`.

## What this app does

- `main.tsx` finds the DOM node and renders React into it
- `App.tsx` shows a small page and calls the backend health route
- `index.css` adds the styling
